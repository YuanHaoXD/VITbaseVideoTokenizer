implement a task-aware injection strategy: for flow-based generation, we introduce diffusion noise corresponding to the time-step  $ t $ ( $ H_{in} = H_{bn} + t\epsilon $); conversely, for understanding, we utilize the clean, deterministic latents to maximize semantic clarity. These processed latents are mapped to the LLM's dimension via a linear projector to yield  $ U_{vis} $. Finally, we concatenate them with text embeddings  $ E_{text} $ to form a unified input stream:

 $$ \mathbf{U}_{\mathrm{i n}}=[\mathbf{E}_{\mathrm{t e x t}},\mathbf{U}_{\mathrm{v i s}}]. $$ 

Dual-Head Decoding. The unified sequence  $ U_{in} $ is processed by the shared LLM backbone. Following standard UMM practices (Xie et al., 2025a; Deng et al., 2025a), we apply a causal attention mask on language tokens and a bidirectional attention mask on visual tokens within the LLM decoder layers. To address the Compatibility of Learning Process between discrete text prediction and continuous image synthesis, we branch the output of the final transformer layer into two specialized heads. The Language Head (autoregressive) functions as a standard linear classifier, predicting the probability distribution over the text vocabulary based on the unified context:  $ \mathcal{L}_{\mathrm{NTP}} = -\sum \log P(\mathbf{y}_i | \mathbf{y}_{<i}, \mathbf{U}_{\mathrm{in}}) $. Simultaneously, the Vision Head (diffusion) serves as a continuous regression head modulated by time-step embeddings via AdaLN-Zero (Peebles & Xie, 2023). This head operates explicitly on the hidden states corresponding to visual tokens ( $ H_{\mathrm{LLM}}^{\mathrm{vis}} $) to predict the flow velocity:

 $$ \mathbf{v}_{\mathrm{p r e d}}=\mathrm{H e a d}_{\mathrm{f l o w}}(\mathrm{A d a L N}(\mathbf{H}_{\mathrm{L L M}}^{\mathrm{v i s}},t_{\mathrm{e m b}})). $$ 

Unified Training Objective. HYDRA is trained end-to-end via a composite objective:

 $$ \mathcal{L}_{\mathrm{t o t a l}}=\mathcal{L}_{\mathrm{N T P}}+\mathcal{L}_{\mathrm{F M}}. $$ 

By sharing the LLM backbone while specializing only the input and output layers, HYDRA achieves rational unification.

#### 2.3. Training Recipe

To cultivate the harmonized nature of HYDRA, we implement a three-stage progressive training strategy. We initialize HYDRA-TOK via reconstruction on large-scale corpora combined with semantic self-distillation against a teacher ViT (Chen et al., 2024b). Subsequent stages incrementally refine the compatibility between understanding and generation (see Appendix D for full data details).

Stage I: Unified Representation Alignment. To resolve the representation divergence at the input level, we freeze the LLM and exclusively tune the vision components. Utilizing 100M image-text pairs, this phase aligns the visual latent space with the linguistic domain, ensuring a coherent unified input representation.

Stage II: Comprehensive Multimodal Pre-training. We unlock all parameters to facilitate harmonized co-promotion within a single unified stream. The model is jointly optimized on a balanced mix of 30M understanding samples and 30M generative samples (strategically filtered from Stage I). This full-parameter update ensures the Compatibility of Learning Process, allowing the two tasks to mutually reinforce each other.

Stage III: High-Quality Instruction Fine-tuning. The final stage focuses on high-fidelity refinement using curated datasets. We employ 3.2M balanced instruction-tuning samples for understanding, alongside 10M aesthetic-filtered images (derived from Stage II) and 6M high-fidelity synthetic images for generation.
