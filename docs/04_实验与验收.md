## 1. 实验总表（按 Phase 枚举）

### Phase 0：校准（1 周，Gate 阻断后续一切）

| Run ID | 内容 | 产出 |
|---|---|---|
| P0-cal-{omni,larp,leanvae,atoken,wan} | 五锚点双跑（官方脚本 + 我们协议），见 03 §4 | 校准报告 |
| P0-sem-cal | SigLIP2 官方权重过我们的 zero-shot 脚本 | 复现其公开 zero-shot（±0.5%），校准语义尺子 |

**Gate P0**：五锚点在我们尺子下排序与公开数字一致；SigLIP2 zero-shot 复现达标。**排序不一致禁止进入任何训练**。
回退：rFVD 排序错→切换 styleganv/videogpt 实现；PSNR 偏差>0.1dB→逐步 diff 预处理（插值核/crop 坐标/值域）。

### Phase B：边界条件探路（OmniTokenizer 仓，2–3 周，与 P1 并行）

消融矩阵（5 组配置 × 2 seed = 10 run + 1 延迟基准）：

| Run ID | attn_mode | fold_mode | 目的 |
|---|---|---|---|
| B-a-full-s{1,2} | full | single(4×) | AToken 式基线 |
| B-a-causal-s{1,2} | causal | single | 官方现状 |
| B-a-tub-s{1,2} | tubelet | single | 发现一移植 |
| B-f-learned-s{1,2} | tubelet | learned(2×+2×) | 发现二移植 |
| B-f-avgpool-s{1,2} | tubelet | avgpool(2×+pool2×) | 第三臂：区分"多阶段"与"有参数" |
| B-latency | 全模式 | — | 17×512² 单前向延迟基准 |

- 配置：官方 stage1 配方缩短（150k 步，lr 1e-3，warmup 20k），ImageNet+UCF 联合，256²，seed 显式设置，**配对**（同 seed 组间同数据序）；
- 评测：UCF val rFVD（主）+ ImageNet rFID + PSNR；
- **Gate PB**（注意：这是边界条件实验，**任何方向的结论都算通过**，不过关的只有"结果不可判读"）：配对 delta 在 2 seed 上符号一致才下结论；符号打架→加第 3 seed；
- 判读输出（写进论文分析章）：tubelet/层级增益在无预训练先验下 {保持/消失/翻转} → 决定论文如何声明结论的适用条件。

| Run ID | 内容 | 配置差异 |

**Gate P1**：P1-base 50k 步 val PSNR ≥26 且 rFID ≤3（**该阈值在拿到 base 前 10k 步曲线后重新标定**——阈值不确定性本身是已知风险）；蒸馏 cosine 距离稳定下降；各扫描臂选优冻结进 `cfgs/uvt_stage1.yaml`。
回退：PSNR 停滞→首查 KL 权重（降一档）；蒸馏与重建互拖→查切分点/λ_dist。

### Phase 2：视频扩展 + 表 1 型消融（3 周）

| Run ID | 内容 |
|---|---|
| P2-pre-{06,04,49}×{iso,zpad} | ADR-3 折叠位置 {(0,6),(0,4),(4,9)} × ADR-4' 锚帧 {隔离,zero-pad} 快筛（各 30k 步，ImageNet+OpenVid 10% 子集）→ 选优冻结 |

| P2-fold-{single,hier}-s{1,2} | 折叠消融（tubelet 固定） |

- 配置：附录 A.4 协议（ImageNet+视频混训 150k 步），配对 seed；
- **Gate P2**：rFVD 配对 delta 排序 tubelet<causal<full 且 hier<single、2 seed 符号一致；tubelet 延迟 ≤ full 的 40%；DAVIS rFVD 绝对值 ≤15（数值级参考，不阻断）。
- 回退：排序不成立→按 §5 复盘协议逐级归因（评测→数据→mask 实现→attention map 目检）；实现无误则如实记录"未复现"并对照 Phase B 结论分析（两处都不成立→Hydra-X 结论可疑；仅主仓不成立→我们的实现或设定问题概率大）。

### Phase 3：蒸馏消融 + 语义评测（3 周）

| Run ID | 蒸馏配置 |
|---|---|
| P3-d-none-s{1,2} | 无蒸馏（纯重建） |
| P3-d-img-s{1,2} | 仅锚帧图像蒸馏（含池化项） |
| P3-d-imgvid-s{1,2} | 图像 + Decompressor(视频教师) ——预期最优 |
| P3-d-imgimg-s{1,2} | 图像 + Decompressor(图像教师逐帧) ——表 2 对照行 |
| P3-d-bidir-s{1,2} | imgvid 配置 + Sem-ViT 双向注意力 ——预期变差 |
| P3-lam-{0.25,1.0} | λ_dist 扫描（imgvid 配置） |

- 评测：zero-shot ImageNet + K400/SSv2 linear probe + CKNNA(DINOv2) + DAVIS 重建（验证蒸馏不伤重建）；
- **Gate P3**：① imgvid 在 SSv2 探针最优且图像探针不降；② bidir 全面退化（复现表 2 结论三）；③ zero-shot：无蒸馏 ≈0 → imgvid >70%（ADR-9 成立判据）；④ 重建退化 <0.3dB。
- 回退：视频项无增益→重跑教师 golden test + 检查 Decompressor 时序对齐；zero-shot <60%→池化蒸馏权重上调/检查 MAP head 初始化。

### Phase 4：三阶段完整训练 + 对标（4 周）

| Run ID | 内容 |
|---|---|
| P4-s1 | Stage1 完整 300k（ImageNet+OpenVid 256² → 后 100k 混入 OpenVidHD+高分图像 bucket） |
| P4-s2-gan / P4-s2-nogan | Stage2 双臂：LARP GAN 配方 vs Gram+DINOv3 感知损失（D15），各 100k |
| P4-s3 | Stage3 调和 50k（统计 z stats→normalize→只训 Sem 侧） |
| P4-inet-only | 纯 ImageNet 版（对齐 Hydra-XTok† 协议，数值级可比的唯一实验） |

**Gate P4**：① P4-inet-only：ImageNet PSNR ≥31.5 / rFID ≤0.25（数据完全对齐，数值级强制，参照 Hydra-XTok† 的 32.96/0.154±10%）；② 全量版 DAVIS rFVD < AToken 的 29.19（必须）、逼近 Wan2.2 的 14.78（尽力）；③ Stage2→3 切换重建退化 <0.2dB（ADR-5 判据）；④ zero-shot/probe 不低于 P3 最优臂。
回退：GAN 崩→nogan 臂顶上（本来就是保险）；②不达→检查数据质量分布后接受并如实报告。

## 3. 统计纪律（所有消融强制，D13）

1. **配对设计**：同一消融的所有臂共用 {模型 seed, 数据序 seed, 初始化}；比较逐对 delta；
2. **≥2 seed**，delta 符号一致才下结论；不一致加第 3 seed，仍分裂则报告"不显著"；
3. 主判据 rFVD/rFID（效应量大），PSNR 只做辅助（Hydra-X 表 1 的 PSNR 效应量 0.04dB 在噪声内）；
4. **红线**：不得为凑趋势在臂间引入 lr/数据/步数差异；不得挑 seed 报最好值（= 学术不端）；负结果照报。

## 5. 复盘协议（每个 Gate 后强制执行，按序归因禁止跳步）

1. 与预期趋势一致？→ 一致：冻结该 Phase 全部 config，写复盘纪要；
2. 不一致：先查**评测**（该指标在校准锚点上重跑还准吗）；
3. 再查**数据**（版本号对吗？抽 10 样本目检；配对的数据序 trace 一致吗）；
4. 再查**实现**（单测全绿？attention map/重建图目检？梯度范数异常？）；
5. 全排除→认定真实差异，如实记录，更新 ADR，必要时邮件作者（zgzaacm@gmail.com）。
